Deploy notes — 2026-05-01
=========================

What changed
------------
- TypeScript: added explicit `ChangeEvent` typings for numerous `onChange` handlers to avoid Vercel build failures.
- UI: fixed `packages/ui` barrel exports and `sidebar` mobile helper.
- DB schema: switched web shim to re-export from `@repo/db/schema` to avoid production re-export issues.

Pre-deploy checks (locally)
---------------------------
1. Run type checks:

   pnpm turbo run check-types --continue=always

2. Build apps:

   pnpm --filter web build
   pnpm --filter realtime build

3. Smoke test the web build locally (optional):

   pnpm --filter web start

Required environment variables
-----------------------------
- Ensure the following envs are present in your Vercel/production environment:
  - VIDEO_KYC_SID
  - VIDEO_KYC_API_KEY
  - VIDEO_KYC_ENV

Deployment
----------
- Create a PR for branch `main` (commits are pushed). After review and merge, Vercel will deploy automatically.
- Verify the admin booking page, verifications page, and helper video KYC flows in staging after deployment.
